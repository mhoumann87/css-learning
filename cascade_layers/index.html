In this example, we are showcasing the power of cascade layers, using @layer. This is a link without a class name. You can see is appears greeb. This is a link with a class og link. It, too, appears green, even thouge we specify the color og .link elements to be blue. While .link has a higher element-level specificty than a, we are setting a color style on a in a higher-precedence @layer

The layer precedence beats the element specificity.

Now, we have an even more specific layer called utilities where wee have a class called pink. This is a link with a class of pink. This specificity wins

Here's what the code looks like (editable). Try removing the typography layer, or adjust the order of the layers to see what happens:
